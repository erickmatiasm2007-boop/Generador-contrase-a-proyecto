# 🔐 Generador de Contraseñas

## 📖 Descripción

El Generador de Contraseñas es una aplicación desarrollada en Python que permite crear contraseñas aleatorias de manera rápida y sencilla. El usuario puede seleccionar la longitud de la contraseña y decidir si desea incluir números, obteniendo como resultado una clave más segura para proteger sus cuentas y datos personales.

---

## 🎯 Objetivo

Desarrollar una aplicación que genere contraseñas aleatorias utilizando Python, aplicando conceptos básicos de programación como funciones, estructuras condicionales, ciclos y librerías.

---

## ⚙️ Tecnologías utilizadas

- Python 3
- Visual Studio Code
- GitHub

---

## 📚 Librerías utilizadas

```python
import random
import string
```

- **random:** permite seleccionar caracteres de forma aleatoria.
- **string:** proporciona letras y números para construir la contraseña.

---

## 🚀 Funcionalidades

- Generar contraseñas aleatorias.
- Seleccionar la longitud de la contraseña.
- Elegir si la contraseña incluirá números.
- Mostrar la contraseña generada en pantalla.

## ▶️ Ejecución del programa

1. Instalar Python 3.
2. Descargar o clonar el repositorio.
3. Abrir el proyecto en Visual Studio Code.
4. Ejecutar el archivo:

```bash
python "generador avanzado.py"
```

5. Ingresar la longitud de la contraseña.
6. Elegir si desea incluir números.
7. El programa mostrará una contraseña generada aleatoriamente.
